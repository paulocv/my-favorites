This example site is for a short course on Git and GitHub for scientists.

The goal is to keep the structure simple:

- one Jekyll site
- one collection for favorite picks
- one shared layout for the pick pages
- one automatic table of contents on the home page

In this example, the topic is aircraft.
For your own project, you can choose any subject you like.
